PASSO 1 — SUBIR CÓDIGO PRO GITHUB (1 vez)
-----------------------------------------
Duplo-clique:  RESOLVER-TUDO.bat

Ele vai:
  • preparar o Git
  • abrir o GitHub para criar o repositório

No GitHub crie repo com nome EXATO:
  troca-copa-2026

Depois copie e cole no terminal (o .bat mostra os comandos).


PASSO 2 — CODEMAGIC (1 vez)
---------------------------
1) https://codemagic.io/apps
2) Add application
3) Repository:  troca-copa-2026
4) Project type:  Flutter
5) Settings → Build:
   • Flutter project path:  .   (só um ponto)
   • Build configuration: codemagic.yaml
6) Code signing → Bundle ID: com.mycompany.trocafigurinha
7) Integrations → App Store Connect (mesma conta Apple)


PASSO 3 — BUILD
---------------
1) Start new build
2) Workflow:  TROCA COPA 2026 — iOS
3) Espere ~20 min
4) iPhone → app TestFlight → instalar
